HeavyMetL is an in-browser software package for determination of relative abundance and label incorporation in 15N-labelled samples analysed by Liquid Chromatography-Tandem Mass Spectrometry, enabling graphical analysis of 15N data without the necessity for command-line interaction.

The quantification algorithm is an iteration on several approaches that have been previously described for 15N, with improved signal-to-noise handling and basic handling for sample-to-sample chromatographic differences.

Processing is performed locally using a multi-threaded JavaScript WebWorker model. The page itself may be saved locally for offline processing and to make permanent edits to the configuration files.

The code is publicly available in two repositories:

  • The quantification software: HeavyMetL
  • A library of general MS-related js code: MSLIB
